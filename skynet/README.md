# Skynet: Robot Factory Simulation

A tiny multithreaded Java simulation: two factions collect random robot parts from a shared factory over multiple days and the winner is the one who can assemble more full robots.

- Two competing factions: "World" and "Wednesday"
- Shared factory produces random parts daily
- Thread-pool driven concurrency
- Winner = faction with more complete robots

## Requirements
- Java 17+ (JDK)
